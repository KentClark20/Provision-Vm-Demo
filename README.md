This is for demo purposes.

You can use the ansible playbooks to automate provisioning of Virtual machines on Openshift and then use the configure web server playbook to demonstrate how to automate webserver configuration.

Step 1: Connect your OPenshift cluster to your Ansible Automation Platform
Step 2: Create a project linking your github repo 
Step 3: Create a Job template that uses create-vm playbook to provision your vms.
Step 4: Create another template that will now configure web services on the vm provisioned.
